# 💰 Sistema Bancário

Projeto desenvolvido em Java com o objetivo de simular operações básicas de um sistema bancário, aplicando conceitos de Programação Orientada a Objetos e lógica de negócios.

---

## 📖 Sobre o Projeto

O Sistema Bancário permite simular o funcionamento de contas bancárias, realizando operações como:

- Abertura de conta
- Depósito
- Saque
- Transferência entre contas
- Consulta de saldo

O projeto foi desenvolvido com foco no aprendizado de regras de negócio e modelagem de classes.

---

## 🧠 Conceitos Aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Métodos e atributos
- Regras de validação
- Organização em camadas
- Estruturas condicionais e repetição

---

## 🛠 Tecnologias Utilizadas

- Java
- IDE (IntelliJ / Eclipse / VS Code)
- Estruturas de Dados
